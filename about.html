About Us

The Project

The Modern Interactive Periodic Table is a web-based tool designed to make learning about the elements fun and engaging. Created in 2025, it features all 118 known elements with interactive filtering, search functionality, and detailed information—perfect for students, educators, and chemistry enthusiasts.

The Creators

Rifatuzzaman: A web developer from Jamalpur, I built this project to combine my love for coding and science. I’m passionate about creating tools that make complex topics accessible, and I enjoy working with HTML, CSS, and JavaScript.

Urfa Hasan Fattah: A designer and science enthusiast from Netrakona, I helped shape the look and feel of this periodic table. I’m driven by a curiosity about how technology can enhance education and inspire others.

Why We Built It

We wanted to create a free, accessible resource that brings the periodic table to life. Whether you’re studying for a test, teaching a class, or just exploring the elements, we hope this tool sparks your interest in chemistry!

How It’s Built

Developed with HTML, CSS, and JavaScript, hosted on Netlify, and styled with a modern gradient design. We used Font Awesome for icons and fetched element data from a custom JSON file to keep it lightweight and interactive.
